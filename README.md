# Defold-3D-Game

A 3D game made with the Defold game engine. The goal of this project is to learn 3D game development and shaders with a simple 3D game.



## The game design

The game is a 3D marble game. It is physics based platformer, where you control a marble with WASD-controls and try to navigate different courses. The camera adjusts based on your velocity making the game feel trippy in a good way.



## Goals

I chose physics based 3D platformer, because it's a great type of a project to learn about 3D game development in terms of 3D forces, 3D collisions, models, level design etc. Overall multiple aspects of 3D game development while keeping the project relatively simple.



The other goal for this project was to learn shaders. That I aimed to learn by using the shaders as much as possible to bring the game to life visually. Shaders are utilized for ambient lighting, visual effects and looks of objects.
