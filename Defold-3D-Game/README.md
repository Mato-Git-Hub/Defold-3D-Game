# Welcome to Defold

This project was created from the "Basic 3D" project template.

The settings in ["game.project"](defold://open?path=/game.project) are mostly set to the default values.

The project contains a bootstrap ["main.collection"](defold://open?path=/main/main.collection) that includes a game object with a camera component and other game objects with three different models.

Check out [the documentation pages](https://defold.com/learn) for examples, tutorials, manuals and API docs.

If you run into trouble, help is available in [our forum](https://forum.defold.com).

Happy Defolding!
